### [DTNS - 每字美句每個你喜歡的瞬間](https://showanne.github.io/DTNS/#/)



> 隨時記錄
> 來自一本書、一句詞、一個讓你感動的瞬間。

> 方便儲存
> 紀錄讓你心動的剎那，隨時回味翻找查詢。

> 便捷分享
> 立即跟親朋好友分享，創造更好的文字風景。

## 主要功能
- 分享牆可以檢視公開的文章、並依據不同的主題文章分類。
- 會員及非會員都可以新增文章及留言問題。
- 會員可以修改及刪除自己的文章。
- 非會員僅能看到有公開分享及非下架的文章。
- 管理員可以新增使用者資料。
- 管理員可以下架文章及回復問題。
